## 🌐 部署到 Vercel

### 方法一：Vercel CLI (推荐)

1. **安装 Vercel CLI**：
```bash
npm i -g vercel
```

2. **登录 Vercel**：
```bash
vercel login
```

3. **部署项目**：
```bash
vercel --prod
```

### 方法二：GitHub + Vercel

1. **推送代码到 GitHub**：
```bash
git init
git add .
git commit -m "Initial commit: Personal Game Panel"
git branch -M main
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
```

2. **在 Vercel 控制台**：
   - 登录 [vercel.com](https://vercel.com)
   - 点击 "New Project"
   - 导入 GitHub 仓库
   - 配置环境变量
